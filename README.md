# someonegg.github.io

Source repository for the `someonegg.github.io` GitHub Pages site.

Site URL: <https://someonegg.github.io>

## What This Repo Is

This is a static site for personal knowledge notes and interactive experiments.  
The main entry is `index.html`, and the site includes:

- `Apps`: interactive tools and playground pages
- `Docs`: technical write-ups and study notes
- `Kids`: Chinese character and math learning pages for children
- `Wiki`: structured indexes for sources, concepts, entities, and queries

## Local Preview

This is a pure static site with no build step required.  
Run a local server from the repo root:

```bash
python3 -m http.server 8000
```

Then open: <http://localhost:8000>

## Deployment

This repository is deployed with GitHub Pages, using the `main` branch as the production source.
