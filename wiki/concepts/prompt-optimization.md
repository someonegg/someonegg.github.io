# Prompt Optimization

## Definition

- 面向提示词（prompt）的系统化优化方法集合，目标是在任务表现、稳定性与泛化之间取得可验证改进。

## Scope

- 包括离散搜索、文本反馈更新、历史轨迹利用、失败驱动修复与回归验证等机制。
- 不等同于一次性 prompt 手工润色，更强调可迭代流程与可复盘证据链。
